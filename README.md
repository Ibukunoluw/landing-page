# landing-page
A responsive landing page for “Shortly,” a URL shortener web app. It uses API integration to shorten links, with copy, delete, and validation features. Built with HTML, CSS, and JavaScript, it is fully responsive and stores links locally.
# 🚀 Shortly – URL Shortener Web App

A modern, responsive URL shortener built using **HTML5, CSS3, and Vanilla JavaScript**.  
This project allows users to shorten long URLs, copy them instantly, and manage saved links — all without a backend.

---

##  Live Demo
https://ibukunoluw.github.io/landing-page/


 Features

- 🔗 URL shortening using API integration
- 📋 Copy-to-clipboard functionality
- 🗑️ Delete saved links
- 🚫 Duplicate link prevention
- 💾 Local storage (links stay after refresh)
- 📱 Fully responsive design (mobile + desktop)
- 🎨 Clean UI with purple-themed design
- ⚡ No page reloads (smooth user experience)

Built With

- HTML5 (Semantic Structure)
- CSS3 (Custom styling + responsiveness)
- Vanilla JavaScript (DOM manipulation + API calls)
- is.gd API (URL shortening service)
